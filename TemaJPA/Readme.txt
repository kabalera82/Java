### Java Persistence API

### Métodos de Persistencia Nativa
    - save()
    - read()
    - update()

### Modelo de Dominio: es el conjunto de clases conceptuales
del mundo real definidas en una aplicación
    - Una DDBB Colegio clases del dominio serían:
       * Alumno * Curso * Asignatura ...

### Modelo DAO: patron de diseño.
    - Si disponemos de la clase Empleado, se creará
    EmpleadoDAO que será la encargada de contener los metodos
    para construir los datos de cada objeto Empleado

### JPA es la API del paquete javax.persistence que se encarga de la conexión.

### POJO Plain Old Java Object. Una clase basica con sus atributos constructores getters y setters
