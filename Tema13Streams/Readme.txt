### STREAM ###
    - Un stream en programacion es una secuencia de elementos
    de datos que se procesan de forma secuencial.
    - No almacena datos: No es una estructura de datos
    sino una abstraccion para procesar datos de una fuente
    - Genera o lee a medida que son necesarios.

    -- Operaciones intermedias Ejemplos comunes
    - filter() -> elimina los que no cumplen con un predicado
    - map() -> Transforma cada elemento del stream
    - sorted() -> Ordena los elementos

    -- Operaciones Terminales: Son las que cierran el steram y producen un resultado
    final
    - collect() -> Agrupa en una coleccion
    - forEach() -> Ejecuta una accion para cada elemento
    - reduce() -> Combina todos los elementos en un solo valor.
